# Spam Email Detection

## Technologies Used
    Python, Pandas, NumPy, NLTK, Scikit, Matplotlib, Seaborn

## Tools Used
    Kaggle /Jupyter Notebook, VS Code, Git & GitHub
    Project Link (if the project is hosted somewhere)
    GitHub Repository

## Instructions to run the project
    Clone the Repository
    git clone https://github.com/Rishi150206/Spam-Email-Detection.git
    cd Spam-Email-Detection

## Install Dependencies
    (Make sure Python (3.8 or above) is installed on your system.)
 
    Install the required libraries using:
        pip install pandas numpy matplotlib seaborn scikit-learn nltk
    
    If a requirements.txt file is available, install using:
        pip install -r requirements.txt

## Add the Dataset
    Download the ‘spam_ham_dataset.csv’.
    Place the dataset file inside the project folder.
    Make sure the filename matches the one used in the notebook.

## Run the Notebook
    Run command: jupyter notebook
    Open the notebook file ‘spam_email_detection.ipynb’ and click Run All Cells.
